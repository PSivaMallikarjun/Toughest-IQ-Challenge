# IQ Test Portal

## Overview
The **IQ Test Portal** is an interactive web-based quiz application featuring some of the world's toughest IQ questions. The portal includes:
- A **multiple-choice quiz** with **9 challenging IQ questions**.
- A **countdown timer** (5 minutes to complete the test).
- **Automatic score calculation** at the end of the test.
- A **simple and responsive UI** built using HTML, CSS, and JavaScript.

## How to Play
1. Start the quiz by opening `index.html` in your browser.
2. Read the question and select the correct answer.
3. Click **Next** to move to the next question.
4. Complete all **9 questions** within **5 minutes**.
5. Your **final score** will be displayed at the end.
